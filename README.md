# Arcaea to Malody Converter谱面转换器v1.0
## 简介
这是一个基于Python的图一乐转谱器，可将Arcaea的aff格式谱面转换为malody谱面，转换时将所有地键转化为tap，天键转化为wipe，hold转化为60宽slide，arc转化为50宽slide。
## 使用方法
双击.py或.exe程序运行，弹出的黑窗口内粘贴aff谱面文件路径（有无引号均可接受），按enter运行后会在转谱器同一目录内生成转换后的谱面文件（1.mc）。需要自行在malody的谱面目录内打包创建子文件夹，不知道怎么放的话看一下其他谱面就可以了，最后注意把音乐文件命名为1.ogg。
## 当前局限性
当前版本不支持转换含6k的谱面，无法转换变速特效（即转换后的谱面始终保持匀速），首尾相接的两段arc会分别转换为两段slide，不会自动合成为一段。此外，对于含有假键（如Aegleseeker）的谱面，转换时会将假键视作真键处理。实现起来其实都不算难，但考虑到这就是个图一乐玩具所以摸了（）
